# simple-crud-django
Simple CRUD con django, utilizando vistas basadas en funciones y autentificacion 


1. Crear entorno virtual `python -m venv nombre-entorno`
2. Instalar dependecias: `pip install -r requierements.txt`
3. Realizar migraciones `python manage.py migrate`
4. Crear superusuario `python manage.py createsuperuser`
5. Correr servidor `python manage.py runserver`

> El usuario administrador de prueba es **admin** y la contraseña **admin123**
